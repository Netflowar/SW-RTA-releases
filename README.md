# SW RTA Companion — Releases

Public release binaries for the **SW RTA Companion** desktop app.

## Download

Check the [latest release](https://github.com/Netflowar/SW-RTA-releases/releases/latest) for the newest `.dmg` (macOS) / `.exe` (Windows).

**macOS** — two flavors:
- `...-arm64.dmg` — Apple Silicon (M1/M2/M3/M4)
- `....dmg` — Intel

Since the app is signed but not notarized, macOS will warn on first launch. **Right-click → Open** the first time, then click "Open" in the dialog. Subsequent launches are normal.

## About

SW RTA Companion is an offline draft & planning tool for *Summoners War* Real-Time Arena — pick/ban assistant, GW planner, box builder, and AI coach.

The source code lives in a private repo. This repo hosts the signed release binaries and serves as the app's auto-update endpoint.

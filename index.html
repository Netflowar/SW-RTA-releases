Install instructions

🍎 macOS (both Apple Silicon and Intel)

1. Open the downloaded .dmg and drag the app to Applications.

2. First launch only: right-click the app → Open → click Open in the warning dialog. After that, it launches normally.

3. The first-launch dance exists because the app is signed but not Apple-notarized. Totally safe — Apple just wants $99/year to skip the right-click step. We haven't paid that yet.

🪟 Windows (coming)

1. Run the .exe installer.

2. If Windows SmartScreen warns, click More info → Run anyway. Same reason — not yet code-signed on Windows.
